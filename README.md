# Twitter Sentiment Analysis using ANN

## Overview
Sentiment analysis model built with Bidirectional LSTM and GloVe embeddings, trained on the Sentiment140 dataset with 1.6 million tweets.

## Results
| Metric | Score |
|--------|-------|
| Accuracy | 76.43% |
| Precision | 76.04% |
| Recall | 87.82% |
| Classes | Positive, Negative, Neutral |

## Tech Stack
- TensorFlow / Keras
- Bidirectional LSTM
- GloVe 100d embeddings
- NLTK preprocessing
- Scikit-learn

## Dataset
Sentiment140 — 1.6 million tweets labeled positive/negative.

## How to Run
1. Open `twitter_sentiment_ann.ipynb` in Google Colab
2. Run all cells in order
